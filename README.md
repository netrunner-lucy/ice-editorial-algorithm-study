# 정통 편집국 알고리즘 스터디

## 스터디 규칙
- 매주 1회 비대면 디스코드 세션 진행
- 스터디 진행 시각 : (미정)
- 진행 사이트 : 백준
- 스터디 진행 언어 : C++
- 한 주 동안 해야 할 일
    - 이론 공부 : 해당 주차 알고리즘 이론 공부하기
    - 문제 풀이 : 공부한 알고리즘에 대해 백준에서 문제풀이. 한 주 할당량은 4문제 (하면서 문제 개수 조정될 수도 있음)
    - 레포에 코드 공유
        ex) 본인 파일명/ 백준 문제 번호_ 문제 이름_ 티어.cpp
    - 파트너 코드 리뷰 : 매주 파트너를 다르게 하여 서로의 코드를 주석으로 리뷰. 리뷰 받은 내용을 바탕으로 코드 수정.

## 기초 멘토링
- C++을 모르는 인원이 많아 초기 주차에 기본 C++ 내용을 기반으로 스터디 진행 예정
- 멘티들은 정해진 분량의 개념을 공부하고 백준 문제를 풀며 복습
- 모르는 부분이 있으면 세션 때 멘토에게 질의응답
- 조 편성
| 멘토 | 박은채 | 김민수 | 진효주 | 임종훈 |
| 멘티 | 이민혁 김정석 | 임형준 전희주 | 장희원 정지민 | 홍선빈 김수인 |

## 코드 업로드 기본 과정
1. 이 저장소를 `clone` 한다.
2. 생성된 저장소에 해당 주차에 맞게 `BOJ_문제번호` 로 폴더를 생성한다.
3. 생성된 폴더에 자신의 소스코드를 업로드한다. ex) `문제이름_자신이름.cpp`
4. commit 규칙을 지켜서 커밋한다.
5. 원본 저장소로 `Pull Request` 를 한다.
6. 다른 사람들의 PR 을 보고 자유롭게 코드리뷰를 한다.
7. PR 후 다음 문제를 풀때 or 다른 스터디원이 커밋 혹은 merge 했을때 `rebase 하기`
